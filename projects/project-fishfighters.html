Fish Fighters

C++ | SFML | Custom 2D Engine

Fish Fighters is a 2D fighting game prototype built on a custom C++ engine using SFML. It includes systems like a State Machine, Entity Manager, and modular UI components.

Features

  • Custom 2D engine with modular architecture
  • Dynamic state management and event system
  • Sprite animations and player input handling
  • Compiled with CMake and versioned using Git
